HeartratePPG¶

Data analytics for time series

  • Photoplethysmography

  • Accelerometer (x, y, z)

Use cases includes

  • heartrate predictions

  • Human Activity Recognition

  • change detection for PPG

Frameworks includes

  • Pytorch

  • CNN

  • LSTM

  • with attention layer

  • with multi-tasks layer

  • singular-spectrum analysis (SSA)
